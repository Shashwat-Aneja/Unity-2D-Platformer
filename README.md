# Unity 2D Platformer

A simple 2D platformer game built in Unity.  
Includes basic player movement, jumping, sprite flipping, collision checks, and a patrolling enemy system.

Designed as a clean and minimal project demonstrating Unity fundamentals for academic and portfolio use.
